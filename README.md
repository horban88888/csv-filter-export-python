
# CSV Filtering & Export Automation (Python)

## What it does
This project contains a Python script that reads a CSV file with customer data (name, age),
filters customers aged 18+, and exports the results into a new CSV file.

## Input
**clients.csv** (UTF-8)
- Columns: `name`, `age`
- Example includes non-English names (e.g., José, Олег)

Example:
```csv
name,age
Ivan,17
Maria,22
John,16
Anna,30
José,19
Олег,45
